The size of a basis

Morphing lemma

Morphing

For any vector space \(\mathcal{V}\) spanned by a finite set \(S\) of vectors (\(S\) is a basis), for any linearly independet set \(B\) of vectors in \(\mathcal{V}\), \(\left| B \right| \leq \left| S \right|\)

Morphing Lemma

Proof outline:

modify \(S\) step by step, introducing vectors of \(B\) one by one, without increasing the size

How?

using the Enchange Lemma

Let \(B=\left\{ \mathbf{b}_1, \ldots, \mathbf{b}_n \right\}\), Define \(S_0 = S\).

Prove by induction on \(k \leq n\) that there is a generating set \(S_k\) of \(\mathcal{V}\) that contains \(\mathbf{b}_1, \ldots, \mathbf{b}_n\), and has size \(\left| S \right|\).

Base case \(k=0\) is trivial. To go from \(S_{k-1}\) to \(S_k\): use the Enchange Lemma:

  • \(A_k = \left\{ \mathbf{b}_1, \ldots, \mathbf{b}_n \right\}\) and \(\mathbf{z} = \mathbf{b}_k\)
  • Enchange Lemma → there is a vector \(\mathbf{w} \in S_{k-1}\) such that $$\text{Span } \left( S_{k-1} \cup \{\mathbf{b}_k\} - \{\mathbf{w}\} \right) = \text{Span } S_{k-1}$$
  • Set \(S_k = S_{k-1} \cup \{\mathbf{b}_k\} - \{\mathbf{w}\}\).

what good is this lemma?

Any basis for \(\mathcal{V}\) is a smallest genereting set for \(\mathcal{V}\)

Let \(B\) be any basis for \(\mathcal{V}\). Certainly \(B\) is a generating set for \(\mathcal{V}\). Let \(S\) be a smallest generating set for \(\mathcal{V}\). Then, by the Morphing Lemma, \(B\) is no bigger than \(S\), so \(B\) is also a smallest generating set.

All bases for a vector space \(\mathcal{V}\) has the same size

They are all smallest generatring sets

Coding the Morphing Lemma

def morph(S, B):
    '''
    Input:
        - S: a list of distinct Vecs
        - B: a list of linearly independent Vecs all in Span S
    Output: a list of pairs of vectors to inject and eject
    '''
    S0 = S.copy()
    A = set([])
    pairs = []
    for z in B:
        w = exchange(set(S0), A, z)
        # S0 = (S0 | {z}) - {w}
        S0.append(z)
        S0.remove(w)
        # A = A | {z}
        A = A | {z}
        pairs.append((z,w))
    return pairs
