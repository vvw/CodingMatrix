Marix-vector equations

Fundamental computational problem

solving matrix-vector equation

  • input: an \(R \times C\) matrix \(A\) and an \(R\)-vector \(\mathbf{b}\)
  • output: the \(C\)-vector \(\mathbf{x}\) such that \(A * \mathbf{x} = \mathbf{b}\)

Solution space of a matrix-vector equation

If \(\mathbf{u}_1\) is a solution to \(A*\mathbf{x} = b\), the solution set is \(\mathbf{u}_1 + \mathcal{V}\) where \(\mathcal{V}=\) Null \(A\). Implications:

  • If \(\mathcal{V}\) is a trivial vector space then \(\mathbf{u}_1\) is the only solution
  • if \(\mathcal{V}\) is not trivial then \(\mathbf{u}\) is not the only solution

\(A*\mathbf{x}=b\) has at most one solution iff Null \(A\) is a trivial vector space
