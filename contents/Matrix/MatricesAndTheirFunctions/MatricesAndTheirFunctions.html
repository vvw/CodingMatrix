Matrices and their functions

Linear functions

Linear functions satisfies two properties. Let \(\mathcal{V}\) and \(\mathcal{W}\) be vector spaces over a field \(\mathbb{F}\). Linear function \(f:\mathcal{V} \rightarrow \mathcal{W}\) satisfies:

  • \(\forall \mathbf{v} \in \mathcal{V}\) and \(\forall \alpha \in \mathbb{F}\), $$f(\alpha \mathbf{v}) = \alpha f(\mathbf{v})$$
  • \(\forall \mathbf{u}, \mathbf{v} \in \mathcal{V}\), $$f(\mathbf{u} + \mathbf{v}) = f(\mathbf{u}) + f(\mathbf{v})$$
Any linear function can be expressed as a matrix-vector multiplication function

Kernel of a linear function

Kernel

The kernel of a linear function \(f\) is, $$\{\mathbf{v}:f(\mathbf{v})=\mathbf{0}\}$$ written \(\text{Ker } f\)

For a funrcion \(f(\mathbf{x} = M * \mathbf{x})\), $$\text{Ker } f = \text{Null } M$$

One-to-one

A linear function is one-to-one iif its kernel is a trivial vector space.

Matrix-vector multiplication function

In this section we will study the relationship between a matrix \(M\) and the function \(f_M: \mathbf{x} \mapsto M*\mathbf{x}\):

  • easy: going from a matrix \(M\) to the function \(f_M: \mathbf{x} \mapsto M*\mathbf{x}\)
  • a little harder: going from the function \(f_M: \mathbf{x} \mapsto M*\mathbf{x}\) to the matrix \(M\)

in studying this relationship, we come up with the fundamental notion of linar function.

from matrrix to function

If \(M\) is an \(R \times C\) matrix over \(\mathbb{F}\), define the function \(f(\mathbf{x}) = M * \mathbf{x}\):

  • domain of \(f\) if \(\mathbb{F}^C\)
  • co-domain of \(f\) if \(\mathbb{R}^C\)

from function to matrix

  • have a function \(f:\mathbb{F}^A \rightarrow \mathbb{F}^B\)
  • want to compute maytix \(M\) such that \(f(\mathbf{x}) = M * \mathbf{x}\)
  • by domain an codomain we now that \(M\) must be an \(B \times A\) matrix
  • finding the entries:
    • write mystery matrix in therms of its columns: $$M = \left[\begin{array}{c|c|c} \mathbf{v}_1 & \cdots & \mathbf{v}_n \end{array}\right]$$
    • use standard generators \(\mathbf{e}_1 = [1,0, \ldots, 0,0], \ldots, \mathbf{e}_n = [0, \ldots, 0,1]\) to determine each column using the known function:
      • \(f(\mathbf{e}_1) = \mathbf{v}_1\) so first column is \(\mathbf{v}_1\)
      • \(\ldots\)
      • \(f(\mathbf{e}_n) = \mathbf{v}_n\) so first column is \(\mathbf{v}_n\)

Inverse

Matrices \(A\) and \(B\) \(\Longrightarrow\) functions \(f(\mathbf{y}) = A * \mathbf{y}\) and \(g(\mathbf{x}) = B * \mathbf{x}\) and \(h(\mathbf{x}) = (AB) * \mathbf{x}\),

Matrix inverse

If \(f\) and \(g\) are functional inverses of each other, we say \(A\) and \(B\) are matrix inverses of each other

  • If \(A\) and \(B\) are matrix inverses of each other, we say \(A\) and \(B\) are invertible matrices
  • denote the inverse matrix of \(A\) by \(A^{-1}\)

If \(A\) and \(B\) are invertible and \(AB\) is defined then \(AB\) is invertible.

why care?

  • A matrix-vector equation is quaranteed to have exactly on solution if the matrix is invertible
  • Algorithms for solving matrix-vector equations \(A \mathbf{x} = \mathbf{b}\) are simpler if we can assume \(A\) is invertible.
  • Invertible matrices play a key role in change of basis
