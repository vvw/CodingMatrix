The function

Informally, for each input element in a set \(A\), a function assigns a single output element from another set \(B\).

In this context:

  1. \(A\) is called the domain of the function
  2. \(B\) is called the co-domain

Formally, a function is a set of pairs \((a,b)\) no two of which have the same first element.

For example: the function with domain \(\{1,2,3, \ldots\}\) that doubles its input is the set: $$\left\{ (1,2), (2,4), (3,6), (4,8), \ldots \right\}$$

Image of an input

The output of a given input is callled the image of that input. The image of \(q\) under a function \(f\) is denotes \(f(q)\)

If \(f(q)=r\), we say \(q\) maps to \(r\) under \(f\). And we write this as \(q \mapsto r\). We wirte: $$f: D \longrightarrow F$$ when we want to say that \(f\) is a function with domain \(D\) and co-domain \(F\).

Image of a function

The image of a function is the set of all images of inputs: $$\text{Im} \, f$$

For example consider the \(\cos (x)\) function; \(\cos : \mathbb{R} \longrightarrow \mathbb{R}\) means the domain es \(\mathbb{R}\) and the co-domain is \(\mathbb{R}\).

However, the image of \(\cos (x)\), \(\text{Im} \, \cos\), is \(\left\{ x \in \mathbb{R}: -1 \leq x \leq 1\right\}\), which is not the same as the co-domain.

Functions between sets

For sets \(F\) and \(D\), \(F^D\) denotes all functions from \(D\) to \(F\), i.e. $$F^D = \{ f \,\, | \,\, f:D \longrightarrow F \}$$

E.g., the set of functions from the set \(W\) of words to the set \(\mathbb{R}\) of real numbers is \(\mathbb{R}^W\)

For finite set, $$\left| F^D \right| = \left| F \right|^{\left| D \right|}$$

Identity function & composition

Identity function

For any domain \(D\), the identity function $$\text{id}_D : D \longrightarrow D$$ maps each domain element \(d\) to itself.

Functional composition

For functions \(f: A \longrightarrow B\) and \(g: B \longrightarrow C\), the functional composition of \(f\) and \(g\) is the function \((g \circ f): A \longrightarrow C\) defined by $$(g \circ f)(x) = g(f(x))$$

Composition associativity

$$h \circ (g \circ f) = (h \circ g) \circ f$$

Functional inverse

Functional inverses

We say functions \(f\) and \(g\) are functional inverses if \(f \circ g\) and \(g \circ f\) are defined and are identity functions.

A function that has an inverse is invertible.

Injective function

Let \(f: D \longrightarrow F\), \(f\) is an injective function (a.k.a one-to-one) if $$f(x) = f(y) \,\,\, \text{implies} \,\,\, x=y \,\,\, \forall x,y \in D,F$$ which means that there are no to distinct elements that have the same image.

Surjective function

Let \(f: D \longrightarrow F\), \(f\) is an surjective function (a.k.a onto) if every element \(y \in F\) has a corresponding element \(x \in D\) such that \(f(x) = y\).

This menas that the function \(f\) maps the co-domain exhaustively. The function \(f\) may map more than one element of \(D\) to the same element of \(F\).

Bijective function

A function \(f\) is known as bijective is it is injective and surjective.

Invertible function

A function is invertible if and only if it's bijective; i.e. one-to-one and onto.
