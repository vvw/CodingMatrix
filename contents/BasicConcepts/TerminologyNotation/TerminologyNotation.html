Terminology & notation

Set

an unordered collection of objects. E.g. {♠, ♣, ♥, ♦}

equality among sets

Two sets are equal if they contain exactly the same elements. A convenient way to prove that \(A\) and \(B\) are equal is to prove that each is a subset of the other. The proof consist in two parts:

  1. a proof that \(A \subset B\) and,
  2. a proof that \(B \subset A\)

Cardinality

If a set \(S\) is not infinite, we use \(\left| S \right|\) to denote the number of elements or cardinality of the set. E.g., the set {♠, ♣, ♥, ♦}, has cardinality 4.

Cartesian product

Given sets \(A\) and \(B\), the Cartesian product \(A \times B\) is the set of all pairs \((a,b)\) where \((a \in A)\) and \((b \in B)\),

For example: for \(A=\{1,2\}\) and \(B=\{♠, ♣, ♥, ♦\}\) $$A \times B = \left\{ (1,♠), (2,♠), (1,♣), (2,♣), (1,♥), (2,♥), (1,♦), (2,♦) \right\}$$

Procedures

Procedure

A procedure is a computation that, given an input, produces an output.

Computational problem

An input-output specification that a procedure might be required to satisfy.
