Linear combinations

Linear combination

Let \(\mathbf{v}_1, \mathbf{v}_2, \ldots , \mathbf{v}_n\) be a set of vector. An expression: $$\alpha_1 \mathbf{v}_1 + \cdots + \alpha_n \mathbf{v}_n$$ is a linear combination of the vectors \(\mathbf{v}_1, \ldots , \mathbf{v}_n\)

The scalars \(\alpha_1, \cdots, \alpha_n\) are the coefficients of the linear combination.

Trivial linear combination

A linear combination is trivial if the coefficients are all zero
